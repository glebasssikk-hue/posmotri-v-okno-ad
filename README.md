# Проектная работа «Посмотри в окно»

## Описание
Данное приложение позволяет «выглянуть» из окон самых разных городов. При загрузке страницы скрипт получает данные из внешнего источника, отрисовывает карточки с видео и подставляет адрес первого ролика в плеер. Скрипт также позволяет искать видео по городам и времени суток.

В рамках этой работы была выполнена верстка интерфейса с использованием:
- CSS Grid для основного лейаута;
- Flexbox для компонентов;
- Кастомных элементов форм (чекбоксы);
- Псевдоклассов `:focus-visible`, `:has()`, `:hover`, `:active`;
- Современных CSS-свойств для позиционирования и оформления.
